# Compétences 3 SAE 401 : Kenzo Fillonneau

## **Synopsis** 

La partie C3 raméne une automatisation qui était pas présente dans le projet de base
Le but était de diffuser du son en multicast via VPN (WireGuard + Tailscale + VXSOUND)
avec toujours un plan B en cas de panne 

Des défi majeures étaient présente mais on été résolus par nos soins

## Initier ## 
Une fois le projet télécharger un 2 scrip bash sont fournis , l'un pour un hôte qui aurait le rôle de serveur et l'autre pour des hôtes qui 
serait définis comme client (il faudrait faire ./soundstram_server i pour initier et de même pour les hôtes).

Le script vous laissera defois la main par exemple pour tailscale (**mettez le même compte que cela soit pour le serv ou alors les clients).

Si les logiciels nécessaire ne sont pas installer le script va le faire tout seul mais il faudrait donc le ré-éxécuter (pour l'instant).

## Explication du script #
